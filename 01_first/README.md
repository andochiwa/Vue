## v-bind

v-bind等被称为指令。指令带有v-，表示它们是Vue提供的特殊特性。它们会在渲染的DOM上应用特殊的响应式行为

v-bind可以绑定一个数据，当鼠标悬浮在上面时，会显示出数据内容

# Computed

计算属性就是当依赖属性的值发生变化时，这个属性的值会自动更新，与之相关的DOM部分也会自动更新

与methods不同的是计算属性是基于它们的响应式依赖进行缓存的，只在相关响应式依赖发生改变时它们才会重新求值
